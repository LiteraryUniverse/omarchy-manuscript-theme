# Manuscript

A theme for those who write by hand before they type — warm parchment, sepia ink, and the quiet hush of a scriptorium.
Every surface evokes a page waiting to be filled: ivory backgrounds carry the warmth of aged vellum, while muted sage,
dusty rose, and soft gold recall the pigments of an illuminated manuscript. The Literary Universe brand magenta appears
as a colophon mark — the scribe's signature, placed with intention.

Designed for writers, readers, and creators who want a workspace that feels like a beautifully appointed study, where
every tool is an invitation to create.

## Color Philosophy

- **Background**: Warm ivory / parchment — the page itself
- **Foreground**: Deep sepia brown — ink on paper
- **Primary accent**: LU magenta (`#A12577`) — the colophon, the publisher's mark
- **Secondary accent**: Dusty sage green — the pressed botanical border
- **Tertiary accent**: Warm gold / amber — gilded initial capitals
- **Error / warning**: Terracotta red — a marginal annotation
